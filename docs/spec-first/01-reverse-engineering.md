# 01 — Reverse Engineering конкурентов КРЕСТ

> Дата: 2026-04-26
> Цель: понять что работает и что НЕ работает у образовательных христианских платформ — взять лучшее, обойти слабости.

---

## Поле игры

КРЕСТ конкурирует одновременно в трёх нишах:
1. **Библейские/евангелизационные приложения** (Bible.com, Alpha)
2. **Православные образовательные платформы РФ** (Исследуйте Писания, ПСТГУ, Школа Православия София)
3. **Платформы для discipleship-сообществ** (Subsplash, RightNow Media)

КРЕСТ — единственная **протестантская русскоязычная платформа управляемого ученичества с no-skip-видео и одобрением лидером**. Это уникальная ниша.

## КОНКУРЕНТ 1 — Bible.com (YouVersion)

### Резюме
Доминирующее библейское приложение мира. 14 млн ежедневных пользователей. Фокус на чтении Библии + готовые планы.

### Анализ по 8 уровням

| Уровень | Что есть |
|---------|----------|
| **Структура** | Лента → План → Чтение → Кнопка "Завершить день" → Streak |
| **Стек** | iOS/Android native + Web (React). Свой бэкенд. CDN для контента |
| **UX** | Минимализм, светлая тема, читабельная типографика, аудио-Библия. Onboarding 30 секунд |
| **Контент** | 10 000+ готовых планов чтения, 150+ анимированных видео (BibleProject), 3 000+ переводов на 1 800 языков |
| **Бизнес-модель** | **БЕСПЛАТНО**, никаких платных тиров. Финансируется Life.Church (евангелистская церковь США) |
| **ЦА** | Универсальная: верующие любых деноминаций. От 13 лет |
| **Уникальность** | Безграничная библиотека планов + социальные функции (Friends, sharing) + Verse of the Day с push |
| **Оценка** | 9/10 техническая, 8/10 UX, 9/10 PMF |

### ✅ Сильные стороны (взять референс)
1. **Streak-система** — сохранение дневной серии чтения (как Duolingo) → главный driver retention
2. **Catch Me Up** — кнопка восстановления при пропуске дней (без чувства вины)
3. **План = micro-commitment** — каждый день только 1 страница, не пугает новичка
4. **Verse Image** — генерация красивых картинок со стихом для соцсетей → виральность
5. **Animated videos** — BibleProject делает 150+ анимаций для аудиалов и визуалов

### ❌ Слабые стороны
1. **Нет лидера и одобрения** — нет связки "ученик → наставник → одобрение"
2. **Нет проверки понимания** — можно "пройти" план не читая
3. **Нет глубины обучения** — это **чтение**, не **ученичество**
4. **Нет российской культурной адаптации** (UI на русском есть, но контент англоязычный)
5. **Нет привязки к местной церкви** — пользователь остаётся изолированным

### 🎯 Что взять для КРЕСТ
- Streak-механика для ежедневной активности
- Catch Me Up — мягкое восстановление после пропуска
- Visual Quote — генерация картинок со стихами для виральности
- Push-уведомления "стих дня" в Telegram

## КОНКУРЕНТ 2 — Alpha Course (Альфа-курс)

### Резюме
Базовый курс знакомства с христианством, проводится в 169 странах с 1977 года. **30+ млн выпускников.** Гибридная модель: видео-эпизоды + малые группы.

### Анализ по 8 уровням

| Уровень | Что есть |
|---------|----------|
| **Структура** | 11 сессий: Episode (15-25 мин видео) → Small group обсуждение |
| **Стек** | MyAlpha hub (веб-портал для лидеров) + Alpha Online платформа для участников |
| **UX** | Сначала Welcome Screen → выбор курса → присоединение к группе → еженедельные встречи |
| **Контент** | Профессиональные видео-эпизоды, обновляются регулярно. Translated в 100+ языков |
| **Бизнес-модель** | **БЕСПЛАТНО** для участников. Лидеры покупают материалы (~$50 пакет). Финансируется HTB London |
| **ЦА** | Skeptics, ищущие, новички в вере. Возраст 18-45 |
| **Уникальность** | **Малая группа** — 5-12 человек, остаётся неизменной все 11 недель → ядро retention |
| **Оценка** | 7/10 технология, 9/10 community-механика, 10/10 PMF для евангелизации |

### ✅ Сильные стороны (взять референс)
1. **Малые группы** — кросс-культурный успех. Люди заканчивают курс ради группы, не ради контента
2. **11 сессий** — оптимальная длина: достаточно для трансформации, не пугает
3. **Каждое занятие = вопрос** ("Кто такой Иисус?", "Как молиться?") — притягивает аудиторию
4. **Лидерская подготовка** — webinars, materials, regional cohorts → масштабируемость
5. **Hybrid model** — онлайн-видео + офлайн-встреча → лучшее обоих миров

### ❌ Слабые стороны
1. **Зависимость от организатора** — без активного лидера курс не запускается
2. **Нет персонального трекинга** — статистика на уровне группы, не индивидуально
3. **Нет gamification** — нет streak, ачивок, прогресса (как в Bible.com)
4. **Сложная инфраструктура для малой церкви** — нужны обученные лидеры, помещение
5. **Нет post-Alpha путеводителя** — курс закончился — что дальше неясно

### 🎯 Что взять для КРЕСТ
- **Малые группы как retention-двигатель** — Telegram-канал у нас есть, но можно **автоматически создавать группу из 5-12 человек, проходящих один блок одновременно**
- Каждый блок = главный вопрос (а не просто "Блок 1: Сотворение")
- Лидерская подготовка — отдельный flow для пасторов-партнёров
- Post-curriculum: что дальше после 6 блоков

## КОНКУРЕНТ 3 — Школа Православия "София"

### Резюме
Бесплатные онлайн-курсы катехизиса от Московского епархиального совета РПЦ. Целевая аудитория — **взрослые, желающие изучить православие систематически**. Аналог КРЕСТ в православной нише.

### Анализ по 8 уровням

| Уровень | Что есть |
|---------|----------|
| **Структура** | Web-платформа sophias.ru → курсы → лекции в видео + текст → итоговый тест |
| **Стек** | Сайт на CMS (вероятно WordPress/MODX), своя система авторизации |
| **UX** | Классический e-learning: список курсов → плеер → следующий урок. Сухо, без gamification |
| **Контент** | Лекции от православных богословов, тексты, тесты. Курсы: Основы православия, Священное Писание, История Церкви |
| **Бизнес-модель** | Бесплатно, государственно-церковное финансирование |
| **ЦА** | Взрослые православные/интересующиеся. Возраст 30+ |
| **Уникальность** | Системность + богословская глубина + признание епархии |
| **Оценка** | 6/10 технология, 5/10 UX, 7/10 контент |

### ✅ Сильные стороны
1. **Систематичность** — структурированная программа, не разрозненные ролики
2. **Авторитет** — благословение епархии = доверие для православной аудитории
3. **Текст + видео** — для разных типов восприятия

### ❌ Слабые стороны (большинство, есть что улучшить)
1. **Нет мобильного приложения** — только сайт. Не подходит для on-the-go обучения
2. **Нет коммьюнити** — учишься в изоляции, нет форума
3. **Нет геймификации** — никаких достижений, прогресса, мотивации
4. **Сухой контент** — лекции в стиле "глаголющая голова", без визуальной обработки
5. **Нет push/email-напоминаний** — пользователь забывает вернуться
6. **Нет роли наставника** — только система → ученик
7. **Не адаптирован под Telegram-аудиторию РФ** (где сидит молодёжь)

### 🎯 Что взять / противопоставить
- Системность — да, КРЕСТ структурирован (6 блоков) → лучше чем хаос
- **КРЕСТ vs София:** мы делаем то же системное обучение, **но с лидером + в Telegram + с retention-механиками** → выигрываем по UX

## КОНКУРЕНТ 4 — Патриаршая программа "Исследуйте Писания"

### Резюме
Запущена в 2022 году по благословению Патриарха Кирилла. Курсы на платформе **GetCourse**. Цель — массовое библейское образование РПЦ.

### Анализ

| Уровень | Что есть |
|---------|----------|
| **Структура** | GetCourse-курсы: видео-уроки → задания → автопроверка |
| **Стек** | **GetCourse** (готовая EdTech-платформа), не своя разработка |
| **UX** | Стандартный GetCourse: список курсов, видеоплеер, чат-помощник |
| **Контент** | "Библия за год", "Новый Завет за год", "Псалтирь за 13 шагов" |
| **Бизнес-модель** | Бесплатно для участников, финансирование РПЦ |
| **ЦА** | Православные миряне, желающие читать Писание систематически |

### ✅ Сильные стороны
1. **Использует готовую платформу** — не тратит ресурсы на разработку
2. **Большой охват** — рассылка по всем епархиям РФ
3. **Простота входа** — нужны email + телефон

### ❌ Слабые стороны
1. **GetCourse** — generic-инструмент, не адаптирован под специфику discipleship
2. **Нет лидера** — массовая модель, без персонального наставничества
3. **Нет Telegram-интеграции** — все взаимодействие через email
4. **Нет малых групп**

### 🎯 Что взять
- Идею "программа за год" (например: "Библия за год" как побочный продукт КРЕСТ для post-curriculum)
- Простота входа (email/Telegram, минимум полей)

## 🎯 СТРАТЕГИЧЕСКИЕ ВЫВОДЫ ДЛЯ КРЕСТ

### Уникальное позиционирование КРЕСТ

> **КРЕСТ = единственная русскоязычная протестантская платформа управляемого ученичества, живущая в Telegram, с персональным лидером и обязательным просмотром видео.**

Никто из конкурентов не закрывает все эти параметры одновременно. Это **defensible moat**.

### 5 главных уроков от конкурентов

1. **От Bible.com — Streak/Catch Me Up:** ежедневная мотивация без чувства вины при пропуске
2. **От Alpha — малые группы из 5-12 человек:** retention растёт когда люди знакомятся друг с другом, не только с лидером
3. **От Alpha — каждый блок = главный вопрос:** не "Блок 1: Сотворение", а **"Откуда взялся мир и я в нём?"**
4. **От Софии — что делать ХУЖЕ:** не быть скучным e-learning без коммьюнити и геймификации
5. **От GetCourse — простота входа:** 1 экран регистрации, без сложных полей

### 3 главные слабости конкурентов = возможности КРЕСТ

| Слабость рынка | Возможность для КРЕСТ |
|----------------|----------------------|
| Все либо чтение (Bible.com), либо лекции (София) — нет **управляемого ученичества** | КРЕСТ занимает эту нишу |
| Нет интеграции с **Telegram** (где сидит русскоязычная молодёжь) | Native Telegram Mini App |
| Нет **post-curriculum** — что дальше после курса? | Путь лидера / горница / 3 курса (КРЕСТ → Путь спасения → Письма Благой Вести) |

### Что КРЕСТ делает правильно (подтверждено анализом)

✅ Ставка на Telegram (vs веб-сайты конкурентов)
✅ Персональный лидер (vs массовый GetCourse)
✅ Структурированные 6 блоков (vs хаос Bible.com)
✅ Бесплатность для студента (стандарт ниши)
✅ No-skip-видео (никто не делает — заставляет реально смотреть)

### Что нужно добавить (на основе RE)

🔄 **Streak + Catch Me Up** (от Bible.com) — ежедневное напоминание читать
🔄 **Малые группы** (от Alpha) — автоматически объединять студентов одного блока в Telegram-чат
🔄 **Каждый блок = вопрос** (от Alpha) — переформулировать названия блоков в вопросы
🔄 **Visual Quote** (от Bible.com) — генерация картинок со стихами для соцсетей
🔄 **Лидерская воронка** (от Alpha) — отдельный onboarding для пасторов-партнёров

### Что НЕ копировать

❌ Не делать "Библию за год" в MVP — это отвлекает от главной ценности (управляемое ученичество)
❌ Не делать gamification ради gamification (точки, очки) — должно быть органично
❌ Не повторять GetCourse-style лекции — мы НЕ образовательная платформа, мы **наставничество**

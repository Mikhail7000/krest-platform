# КРЕСТ — История изменений приложения

Все заметные изменения платформы «Крест» — в одном файле, по версиям.
Описания — для человека (что появилось/поменялось для учеников, кураторов и руководства), не техническим языком.

Формат: версия → дата → список изменений. Свежие версии сверху.

---

## v1.9 — ИИ-тренажёр «Учиться вместе с ИИ» · 2026-06-24

- **Новый раздел в тренажёре местописаний — чат с искусственным интеллектом.** Наверху тренажёра — подсвеченная кнопка «✨ Учиться вместе с ИИ». По нажатию открывается живой чат: ИИ ведёт квиз по стихам блока.
- **Типы заданий в чате:** даёт текст стиха → просит назвать ссылку; даёт ссылку → просит привести текст; «вставь пропущенные слова»; «из какого блока этот стих». Проверяет ответы мягко по эталону, хвалит, тактично поправляет.
- **Ответ голосом и видео-кружком (фаза 2).** В чате можно отвечать не только текстом, но и записать 🎤 голос или 🎥 видео-кружок — запись распознаётся и уходит ИИ как ответ. Полоса записи с таймером, превью кружка и отменой.
- ИИ заземлён строго на стихи текущего блока — ничего не выдумывает.

## v1.8 — Скорость приложения · 2026-06-24

- **Мгновенная навигация.** Экраны открываются без ожидания: статус блока, дашборд, рейтинг и лента показываются сразу из кэша и тихо обновляются в фоне. Раньше на странице урока статус грузился несколько раз подряд — теперь один общий запрос.
- **Мгновенная отметка молитвы.** Галочка «помолился» загорается сразу при клике (отправка идёт фоном). Счётчик дней и «день закрыт» по-прежнему берутся с сервера — корректность не страдает.
- **Скелетоны загрузки** вместо текстов «Загрузка…» и «…» — приложение ощущается быстрее.
- **Ускорение базы данных** под рост числа учеников (индексы для проверки закрытия дня).

## v1.7 — Часовые пояса и согласованность «дня» · 2026-06-24

- **Критическое исправление: дни теперь закрываются.** Раньше из-за стиха-притчи, который сдаётся только аудио, гейт местописаний был невыполним — и день не закрывался ни у кого в стартовом блоке. Исправлено.
- **День закрывается в 00:00 по часовому поясу ученика** (а не по UTC). Следующий день и повтор практик открываются строго после полуночи по его времени.
- **Согласованность «день закрыт»** в приложении, в админ-панели и у куратора — везде одинаково.
- **Уведомление куратору**, когда его ученик закрыл день (только про своих учеников).
- **Напоминания приходят в 20:00 по местному времени** каждого ученика (а не по Бали), с ротацией текстов.
- На главном экране — крупный прогресс «Закрыто дней X/7»; на экране блока — пояснение, когда откроется следующий день.

## v1.6 — Тренажёр, темы и интерфейс урока · 2026-06-24

- **Порядок практик блока:** сверху — обязательные для закрытия дня (фото креста, молитва, местописания, пересказ), ниже — «по желанию» (тренажёр, эмоции).
- **Зелёная подсветка** карточки практики, закрытой сегодня (вся карточка, не только бейдж).
- **Розовую тему можно включить прямо с главного экрана** (кнопка темы: солнце → луна → звёзды → роза).
- Карточка «Вводный модуль» теперь следует выбранной теме (в тёмной — тёмная, в розовой — розовая), а не «горит белым».

## v1.5 — Трекинг и связь с учениками · 2026-06-24

- **Тестовый аккаунт «Крест Капсула Тест» — вне рейтинга**, всегда в самом конце списка (не занимает первое место).
- **Клик по нику ученика в трекинге открывает с ним чат в Telegram** напрямую.

## v1.4 — Веб-панель администраторов и кураторов · 2026-06

- **Веб-дашборд /panel** со входом через Telegram: статистика, управление учениками, заявки.
- **Раздел «Активность»** — таблица учеников с отметкой активности по дням; отчёт одной кнопкой и копирование отчёта текстом.
- **Панель куратора (ограниченная):** куратор видит только своих учеников.
- **Управление ролями** в панели: перевод между ученик ↔ куратор ↔ администратор, с историей решений (кто и когда принял).
- **Сортировка** списков по любому заголовку; **прямой выбор куратора** в строке ученика.
- **Заявки на доступ:** видны и решаются в панели; при одобрении показывается статус доставки Telegram-уведомления.
- **Лента сообщества:** эмоции и свидетельства; фото показываются целиком, открываются по тапу.
- Раздел «Пол» убран из профиля (тема выбирается отдельно).

## v1.3 — Telegram-бот: кураторы и уведомления · 2026-06

- **Команды бота** для кураторов и админов: добавление куратора, массовая привязка учеников (`/attach`), список кураторов с «ожидают входа», меню по ролям.
- **Уведомления администраторам** о новых учениках и заявках.
- Владелец платформы (защищённый аккаунт) видит скрытых тестировщиков.
- Толерантность бота к лишним слешам в командах.

## v1.2 — Дневная модель сдачи и гейт · 2026-05 … 2026-06

- **Единая дневная модель «День N/7»:** чтобы закрыть день, нужно выполнить 4 практики — написать крест (фото), помолиться, пересказать блок (аудио), сдать местописания (видео-кружки).
- **Гейт блока:** 7 закрытых дней открывают следующий блок (forward-only).
- **Местописания** — отдельная ежедневная практика (видео-кружок) с проверкой; **пересказ** — аудио своими словами.
- **Иерархия экзаменов:** block-gate в каждом блоке, промежуточный экзамен после блока 5, финальный — у администратора.
- **Тестовый режим** для тестировщиков: ускоренные таймеры (без ожидания дней), безлимитные пересдачи.
- Квиз убран из приложения — день держится на 4 практиках.

## v1.1 — Тренажёр, трекинг и геймификация · 2026-05 … 2026-06

- **Тренажёр местописаний:** учебный режим перед сдачей — карточки, «вставь пропущенные слова», викторина по ссылкам, запись вслух; избранные стихи.
- **Рейтинг-лидерборд:** аватарки, очки, серии «дней подряд», ачивки.
- **Ежедневная активность и Telegram-напоминания**, прогресс на главном.
- **Темы оформления:** светлая, тёмная, «звёзды», розовая (розовая — автоматически по полу).

## v1.0 — Базовая платформа · старт

- **Telegram Mini App + браузерная версия** платформы управляемого ученичества по курсу «Крест».
- **Онбординг:** язык → страна → город → куратор → данные. Старт — только Бали; остальные локации открываются по мере назначения кураторов.
- **Курс из 10 блоков** + вводный модуль; уроки с видео (Kinescope, без перемотки), конспектами и ресурсами.
- **Роли:** ученик, куратор, администратор, супер-администратор; защита владельца платформы от разжалования.
- **Видимость как маркер прогресса:** ученик видит свою группу, после сдачи курса — шире.
- **Лендинг** платформы.
- Технологии: Next.js + Supabase + Telegram Bot API + Kinescope + Anthropic (ИИ) + Deepgram (распознавание речи), деплой на Vercel.

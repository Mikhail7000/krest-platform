## 🎵 Стратегия заливки контента (согласовано)

**В Supabase Storage (bucket `block-resources`):**
- m4a — молитвы (все блоки)
- pdf — молитвы текстом, гайды («Эпоха пятницы»), материалы папки «Важно» для кураторов
- доп. изображения

**Структура путей:** `block-resources/01-maly-krest/audio/...`, `.../pdf/...`, `.../guides/...`

**Доступ:** public read для авторизованных студентов, upload только admin. Раздел «Важно» — RLS только для curator/admin.

**В БД (миграции):**
- Транскрипции → `lessons.transcript_md`
- ДЗ (9 пунктов) → таблицы `assignments` + `submissions`
- Местописания → `bible_verses` (уже есть в схеме)

**НЕ в Storage:** видео (Kinescope), структурированный текст.

**Скрипт заливки:** напишу одноразовый `scripts/upload-resources.ts` на Этапе Б шаг 2. Пройдёт по папке материалов, зальёт в Storage, заполнит таблицу `block_resources` ссылками. Использует service_role key.
